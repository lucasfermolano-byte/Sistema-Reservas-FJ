# Sistema Integral de Gestión - Software FJ

## Descripción
Sistema orientado a objetos desarrollado en Python para la gestión de clientes, servicios y reservas sin uso de bases de datos.

## Características
- Registro de clientes
- Gestión de servicios
- Creación de reservas
- Manejo de excepciones
- Registro de logs
- Programación orientada a objetos

## Tecnologías utilizadas
- Python
- GitHub

## Estructura del proyecto
- Cliente
- Servicio
- Reserva
- Sistema
- Logger
- Excepciones

## Ejecución
Ejecutar el archivo:

```bash
python src/main.py
```
